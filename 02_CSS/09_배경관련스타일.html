배경관련 스타일(background)

background-color: 배경색을 지정할 수 있는 스타일 속성

웃을 수 있을 때 언제든 웃어라. 공짜 보약이다.

배경으로 이미지 넣기

        [ 표현법 ]
        선택자 {
            background-image: url(이미지경로);
            background-repeat: repeat / repeat-x / repeat-y / no-repeat
            background-size: auto / contain / cover / 숫자(px/%)
            background-position: left / right / center;
                                 top / bottom / center;
            background-attachment: scroll / fixed;
        }

backgournd-clip: 배경을 적용하고자하는 범위를 지정할 때 사용

        [ 표현법 ]

        선택자 {
            backgournd-clip : border-box / padding-box / content-box
        }
