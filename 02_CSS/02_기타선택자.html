    3. 동위(같은레벨) 선택자

    동위 관계에 있는 특정 요소를 선택할 때 사용

            [ 표현법 ]
        
        - a요소 뒤에 있는 b요소 딱 하나만을 선택
        a + b {
            스타일속성 : 값;
        }
        
        - a요소 뒤에 있는 b요소를 선택
        a ~ b {
            스타일속성 : 값;
        }

    4. 반응 선택자

    사용자의 움직임에 따라서 요소를 선택

            [ 표현법 ]
        - 해당 요소를 클릭하고있을 때 선택
        선택자:active{
            스타일속성 : 값;
        }

        - 해당요소(border)에 마우스커서가 올라가는 순간 선택
        선택자:hover{
            스타일속성 : 값;
        }

    5. 상태 선택자

            요소의 상태에 따라 선택을 할 수 있는 선택자

        선택할 수 있는 상태의 종류
        1. checked 상태
        2. focus 상태 
        3. enabled 상태
        4. disabled 상태
    
            2) focus가 맞춰진 input 요소 선택

        선택자:focus{
            스타일속성 : 값;
        }
    입력해보아요 : 
            3) enabled : 활성상태

        4) disabled : 비활성상태
